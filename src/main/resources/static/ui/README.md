#### 项目介绍

<p style="padding:10px;"  width="90%">

基于Layui的后台管理系统模板，扩展Layui原生UI样式，整合第三方开源组件，提供便捷快速的开发方式，延续LayuiAdmin

的设计风格，持续完善的样式与组件的维护，基于异步Ajax的菜单构建，相对完善的多标签页，单标签页的共存，为使用者提

供相对完善的开发方案，只为成为更好的轮子，项目不定时更新，建议 Star watch 一份

</p>
